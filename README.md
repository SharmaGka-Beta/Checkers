# **Checkers**

A GUI-based checkers game made in python.

### **Key Features**

1. Two player turn based gameplay  
2. Captures and king mechanic  
3. Highlighting possible moves  
4. Interactive GUI board  
5. Saving and loading games  
6. Background music  

### **Resources Used**

1. Python 3 - Core game logic  
2. Tkinter - GUI elements  
3. Pygame - Background music 

### **Installation**

```bash
git clone https://github.com/SharmaGka-Beta/Checkers.git
```


```bash
cd Checkers
```

```bash
python3 -m venv venv        #Linux/Mac
python -m venv venv         #Windows
```

```bash
source venv/bin/activate    #Linux/Mac
venv\Scripts\activate       #Windows
```

```bash
pip install -r requirements.txt
```

```bash
python3 main.py     #Linux/Mac
python main.py      #Windows
```

Make sure to add the virtual environment inside the main directory and run the code from inside the main directory.
Installation of Pygame is required only for running the background music without which the game will still run seamlessly.

### **Main Files**

Welcome Screen - GUI of the Main Menu  
Board - GUI of the game  
Game Logic - Core logic of the game  
Game State - Updating GUI
